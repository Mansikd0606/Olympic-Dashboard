# 📊 Olympic Data Analysis Dashboard (Power BI)

📌 Project Overview

This project focuses on analyzing historical Olympic data using Power BI to uncover insights related to medal distribution, country performance, and sports trends.
The dashboard provides an interactive way to explore Olympic achievements across different countries, sports, athletes, and years.

🎯 Objectives

- Analyze total medal distribution across countries
- Compare performance by sport category
- Track medal trends over time
- Enable user-driven filtering using slicers
  
🛠️ Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation (Power Query)

📈 Key Measures Created

- Total Medals
- Total Gold Medals
- Total Silver Medals
- Total Bronze Medals
These measures were created using DAX to dynamically calculate medal counts based on filters and slicers.

📊 Dashboard Visuals

- Total Medals by Country → Compare top-performing nations
- Total Medals by Sport → Identify dominant sports categories
- Year-wise Analysis (Tile/Slicer) → Track performance over time
- Athlete Slicer → Drill down into individual athlete performance
  
🔍 Key Insights

- Certain countries consistently dominate across multiple Olympic years
- A few sports contribute the majority of medals
- Medal distribution varies significantly across years and regions
- Athlete-level filtering helps identify top performers
  
🚀 Features

- Interactive dashboard with slicers
- Dynamic filtering (Year, Athlete)
- Clean and beginner-friendly UI
- Real-time measure calculations using DAX

✅ Conclusion

This project demonstrates how Power BI can transform raw Olympic data into meaningful insights through interactive visualizations. By creating key measures and using slicers effectively, the dashboard allows users to explore medal trends across countries, sports, and years.

As a beginner-level project, it highlights strong fundamentals in:

- Data modeling
- DAX calculations
- Visualization design
- User interactivity

Overall, this dashboard serves as a solid foundation for more advanced analytics projects and showcases the ability to turn data into actionable insights using Power BI.
